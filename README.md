# go-game
An SDL based GoLang game.

## How to run
  1. Make sure you have [Go installed](https://golang.org/dl/)
  2. Make sure you have [SDL2 installed](https://www.libsdl.org/download-2.0.php)
  3. Execute the 'execute' file by typing (on mac/linux):
  ```
  chmod +x execute
  ./execute
  ```
